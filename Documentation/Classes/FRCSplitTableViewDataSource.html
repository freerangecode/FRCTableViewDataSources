type

The type of the split.

@property (nonatomic, assign) FRCSplitTableViewDataSourceType type

Discussion

The type of the split.

  • FRCSplitTableViewDataSourceTypeSection Splits each child table view data source into separate sections
  • FRCSplitTableViewDataSourceTypeRow Makes this data source responsible for once section and joins each child table view data source one after the other inside this section.
