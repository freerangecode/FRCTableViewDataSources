cellClass

This is the class to use for cells of this data source.

cellClassAtIndexPath:

To enable a data source to have different cell classes for different index paths.

- (Class)cellClassAtIndexPath:(NSIndexPath *)indexPath

Parameters

indexPath

The index path in the co-ordinate space of the data source.

Return Value

The class of the cell to use.

Discussion

This method returns the class in the property cellClass. Subclasses should override for different results.

objectAtIndexPath:

To get the associated object from the data source for the given index path. By default this returns the index path, but subclasses should return the correct object to use.

- (id)objectAtIndexPath:(NSIndexPath *)indexPath

Parameters

indexPath

The index path in the co-ordinate space of the data source.

Return Value

The representing object.

Discussion

If the cellClass conforms to FRCTableViewCellObjectConfiguration, it is this object that will be given to the cell when configureWithObject: is called.
